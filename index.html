The main challenges with Excel based models

Limited Data Storage

As the data used in models increase, excel has difficulty in storing and analysing large groups of data sets resulting in extremely slow models and/or errors.

Single User Access

When multiple users try to open the same file it causes glitches as to which is the final version to be saved.

Vulnerable

Excel based models are susceptible to Human Error. It can be difficult to troubleshoot or test to find out where the issue is.

Difficult to Expand Scope

As more analysis/calculations is added to a model, it becomes difficult to expand the logic of Excel based models.

There are better ways to store, organize, and analyse data. A big part of joining this course is to understand the different options available.
